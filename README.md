# EnterpriseLab
An independently designed packet tracer lab utilizing several networking principles. 

The topology includes:
10 VLANs + voice VLAN
Inter-VLAN routing on redundant L3 core switches
HSRP for gateway redundancy
Multi-area OSPF across simulated branch networks
EtherChannel at Layer 2 and Layer 3
STP optimization with controlled root 
DHCP relay with centralized DHCP services and manual fail-over
Security hardening including DHCP snooping, port security, and unused port shutdown
